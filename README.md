# Rule-Based Chatbot

## Project Overview

This project is a simple Rule-Based Chatbot developed using Python. The chatbot responds to user queries based on predefined rules and pattern matching techniques. It uses conditional statements (`if-elif-else`) to identify user inputs and provide appropriate responses.

The project demonstrates the basic concepts of Natural Language Processing (NLP), conversation flow, and decision-making logic.

---

## Objective

To develop a chatbot that can:

* Interact with users through text-based conversation.
* Identify user queries using predefined rules.
* Provide relevant responses.
* Demonstrate basic AI and NLP concepts.

---

## Features

* Greeting responses (Hello, Hi, Hey)
* Answer basic questions
* Display current date
* Display current time
* Respond to thank-you messages
* Explain its purpose
* Exit conversation on user command
* Handle unknown queries gracefully

## Working Principle

1. The chatbot accepts user input.
2. The input is converted to lowercase.
3. Pattern matching is performed using conditional statements.
4. If a matching rule is found, the corresponding response is displayed.
5. If no rule matches, the chatbot returns a default response.
